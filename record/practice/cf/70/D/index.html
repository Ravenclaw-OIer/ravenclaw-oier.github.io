CF70E. Information Reform

Posted on 2020-08-27

Paraphrased Problem Statement

Given a tree $T$, you need to make, at the cost of $k$ dollars, some nodes into signal emitters. For each node, you need to pay $w_d$ dollars if the distance between it and the nearest emitter is $d$. Minimize the total money spent.

Solution

This problem is obviously a DP on trees problem.

Let $f(i, u)$ denote the best solution when $u$ and its subtree get their signal from $i$. The answer is thus $\max^{n}_{j=1}f(j, i)$.

For $v \in \operatorname{subtree}(u)$, there are two choices:

  • Get signal from $i$, then we need to pay $f(i, v) - k$ dollars for $v$. (Update is performed after recursively solving for $v$.)
  • Or, get signal from another node $j$, where $f(j, v)$ is minimized for all $1 \leq j\leq n$. We can easily store $j$ for later use after weâ€™ve done with the recursive calculations.

Printing the path is trivial.
