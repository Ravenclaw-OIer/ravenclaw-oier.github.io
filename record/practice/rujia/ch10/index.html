Examples

11582. Colossal Fibonacci Numbers!

Take $n=5$ as an example. We define $F$ as the Fibonacci sequence after taken modulo $n$.

With some brute force we get

We can find that $F_{20}=F_0$(assuming that $F$ is $0$-based), $F_{21}=F_1$, and so on, thus, the sequence is periodic. Since there can be no more than $n$ different remainders, $F$ will repeat before we meet its $n^2$-th element.

So we only need to calculate $F(i)$ for all $0 \le i \le n^2$, and just calculate which place will $F(a^b)$ be in the repetetion.

The overall time complexity is $O(n^2+T\log b)$. Since $T$ is very small, the time complexity is $O(n^2)$.

12169. Disgruntled Judge

Brute force for $a$ and $b$, then calculate the whole sequence, break off when noticed it is unacceptable.

The complexity is supposed to be $O(n^2T)$, however, the constant is so small that it can pass test cases where $n$ can be up to $10^4$ with ease.

10820. Send a Table

Since we can derive $f(kx, ky)$ from $f(x,y)$, we only need to keep $ f(x,y)$ with $(x,y)=1$ in the table, as other $(x,y)$ can be written as $(dx_0,dy_0)$ where $d=\gcd (x,y)$.
