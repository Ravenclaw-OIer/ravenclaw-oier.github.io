Contest rating was changed $\color{green}+36$.

Problem D. Handstand 2

It is harder to use constructive algorithms actually we define $f(i, j)$ as the number of integers within the range $[1,n]$ that starts with $i$ and ends with $j$. This can be trivially calculated by iterating through all integers within the range.

Then, for given $i$ and $j$, there are a total of $f(i, j) \times f(j, i)$ pairs that satisfies the condition in the problem. Thus, the answer is $\sum_{i=1}^{9}\sum_{j=1}^{9}f(i,j) \times f(j,i)$.

The overall complexity is $O(n+k^2)$, where $k$ is constant $9$. Thus, the time complexity of this solution is $O(n)$.

Problem E. Flatten

It is easy to observe that $K=A_iB_i$ holds for all $1 \le i \le n$. In order to minimize $B$, $K$ must be minimized, and we can see that when $K=[A_1,A_2, \ldots, A_n]$, $K$ is minimized, and the $B$ array is minimized.

However, $K$ is very large and it well exceeds the limit of $\texttt{long long}$. Thus, we will store $K$ in a factorized manner, like $x=p _ { 1 }^{ {\alpha} _ {1} } p _ { 2 } ^ { { \alpha } _ { 2 } } p_ { 3 } ^ { { \alpha } _ { 3 } }\cdots$.

Maintain an array $f$ , and update $f$ for the largest $\alpha$ when factorizing all of its elements. And we will get $K$ in the factorized form.

We can then calculate $K$ modulo $10^9+7$. After that, use multiplication inversion to calculate the value of $\sum_{i=1}^{n}\dfrac{K}{a_i}$, which is the answer.

The overall time complexity is $O(n\sqrt{a})$, where $a$ denotes the largest element in the $A$ array.

Problem F. Tree and Constraints

The basic idea is inclusion-exclusion. Instead of considering “how to satisfy all constraints”, let’s consider “how to violate at least one constraint”.

Let $S$ be the set of all constraints. Take $s \subseteq S$. Let $c(s)$ be the number of edges involved in these constraints, it can be trivially calculated provided that the path between any pair $(i,j)$ is pre-calculated. The number of ways to violate all constraints that belongs to $s$ will be $f(s)=2^{n-1-c(s)}$ (all edges involved must be colored white while the edges that are not involved can be colored arbitrarily).

Then, the final answer will be $\sum_s-1^{|s|}f(s)$, using the inclusion-exclusion principle. The over all time complexity is $O(n^32^m)$.
