Problem E. Crested Ibis vs Monster

Let $dp_i$ be the minimum cost to cut $i$ points of health from the monster.

The answer is $dp_h$.

Problem F. Silver Fox vs Monster

We may observe that in order to kill monster $i$, there must be at least $\lceil\dfrac{h_i}A\rceil$ bombs that can harm it. Optimally, we put the bomb as far as possible, so for monster $i$, we put a bomb at $x_i+2d$ to harm it.

Iterate through all monsters, use a segment tree to keep the harm it has received, then binary search to find the furthest one the bomb can reach, and perform an interval add operation on the segment tree.

The overall time complexity is $O(n \log n)$. If you use sqrt decomposition then $O(n \sqrt{n})$
